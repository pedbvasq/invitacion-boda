Paso a paso

Como obtener el link de Google Photos

  1. Abre Google Photos

    Ve a photos.google.com

  2. Crea un nuevo album

    Haz clic en "Albumes", luego en "Crear album" y ponle un nombre claro.

  3. Activa la colaboracion

    Dentro del album, usa el boton de compartir y habilita la opcion para que otros agreguen fotos.

  4. Copia el link

    Selecciona "Obtener enlace" o "Copiar enlace".

    El formato normalmente se vera asi: https://photos.app.goo.gl/xxxxx

  5. Pegalo en este panel

    Guarda el enlace para que quede disponible en la invitacion.

Importante: confirma que la colaboracion este activa para que los invitados puedan subir sus fotos sin problemas.
