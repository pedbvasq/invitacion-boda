Paso a paso

Cómo obtener el enlace de Google Photos

  1. Abre Google Photos

    Ve a photos.google.com

  2. Crea un nuevo álbum

    Haz clic en "Álbumes", luego en "Crear álbum" y ponle un nombre claro.

  3. Activa la colaboración

    Dentro del álbum, usa el botón de compartir y habilita la opción para que otros agreguen fotos.

  4. Copia el enlace

    Selecciona "Obtener enlace" o "Copiar enlace".

    El formato normalmente se verá así: https://photos.app.goo.gl/xxxxx

  5. Pégalo en este panel

    Guarda el enlace para que quede disponible en la invitación.

Importante: confirma que la colaboración esté activa para que los invitados puedan subir sus fotos sin problemas.
